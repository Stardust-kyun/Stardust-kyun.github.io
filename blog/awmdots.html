AwesomeWM Dotfiles

07/20/23

A list of dotfile repositories for awesomewm. If you have any suggestions, requests, or concerns, contact me.

Amarena - Elenapan

One of the greats. Wonderfully designed widgets, a unique bar, and one of the best user configurations made. A pioneer of many trends, and perhaps one of the first rices to ever be considered a desktop environment.

The Glorious Dotfiles - Manilarome


A great rice for function despite its divisive design. Many common widgets like the lockscreen, dock, and notification center were popularized here.

awesome-config - Worron

Old but gold. An incredibly featureful desktop, with many custom widgets and features still unique to this day.

crylia-theme - Crylia

A desktop environment with a beautiful theme and wonderful user experience. Contains great, easy to understand examples for the many components of a desktop.

KwesomeDE - Kasper

Perhaps the closest to a fully featured desktop environment anyone has made, complete with its own applications. Contains many unique features such as pywal support, a setup tool, gui user configuration, and more.

Sakura - Stardust-kyun

Simple desktop environment with semi-modular widgets. It aims to have as many features as possible while still being readable and slim. Contains a theme switcher and gui user configuration. Great resource for beginners.

Crystal - Chadcat

Beautiful desktop environment with simple widgets. Widgets are easy to read and understand, with great examples for more difficult ones like weather, notifications, run launchers, etc.

SlimeOS-awesome - SkyyySi

A featureful desktop environment with fully complete desktop icons, window tabs, and countless unique widgets. Setting it up will probably be painful.

Syntax - Sammyette

Incredibly well made desktop environment from the creator of Hilbish. Recreation of a windows theme in awesome, made worse with a dark theme.

gwileful - Gwynsav

Beautiful rice with elegant design and improved stability. Created by a self-described xserver terrorist. Gwileful selling points: not a digital nuke.
